# Penguin Ice Bricks (Definitive Edition)

A 3D web-based recreation of the classic "Penguin Ice Bricks" board game. Built with Three.js and Vite.

## ğŸ® How to Play
1. **Objective**: Don't let the penguin fall!
2. **Gameplay**: Click on the ice blocks to break them.
3. **Game Over**: If the block under the penguin breaks or loses support, the penguin falls and the game ends.
